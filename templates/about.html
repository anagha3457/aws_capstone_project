About Us

FaveBy is a cloud-based AI-Driven Personalized Marketing Platform system that uses machine learning to deliver personalized marketing messages to customers in real time. Built on AWS, it analyzes customer behavior and preferences to automate targeted campaigns, helping improve user engagement, conversion rates, and customer retention.

Amazon EC2 runs the marketing models, while customer data and campaign metrics are stored in DynamoDB for fast and scalable access. AWS SNS is used to send personalized notifications through email, SMS, or push messages. Security and access control are managed using AWS IAM, ensuring safe and authorized system usage. Overall, the platform enables scalable, secure, and data-driven personalized marketing.

Tech Stack Details

  • Frontend: HTML, CSS, JavaScript for responsive and interactive UIs.
  • Backend: Python Flask for robust API and application logic.
  • Cloud Infrastructure: Amazon Web Services (AWS) ensures scalability and reliability.

Why AWS?

AWS offers the broadest and deepest set of cloud services, making it the perfect choice for learning and deploying modern applications. Our projects utilize:

  • IAM (Identity and Access Management): Securely managing access to services and resources.
  • DynamoDB: A fast and flexible NoSQL database service for any scale.
  • Amazon SNS (Simple Notification Service): Fully managed messaging for microservices, distributed systems, and serverless applications.
  • Amazon EC2 (Elastic Compute Cloud): Secure and resizable compute capacity for virtually any workload.
